# Roadmap — 版本路线图

> 原则：小步快跑，每个版本可独立运行。不追求一次性完成所有功能。

## 版本哲学

- **Patch (v0.0.x)** — bug 修复、文档、小型增量。
- **Minor (v0.x.0)** — 一个完整可用的模块或里程碑。
- **Major (v1.0.0)** — 四大模块集成完整，可作为日常主力系统。

状态图例：✅ 完成 / 🚧 进行中 / 📋 规划中

## 规划版本

### v0.0.2 — Body 模块扩展 📋
- 体重记录的增删改查 (CRUD) 完善
- 时间范围查询（按日/周/月）
- 数据导入/导出 (CSV)
- 基础统计（均值、趋势）

### v0.1.0 — Body 模块 MVP 📋
- 前端体重记录页面 (Vue 3)
- 体重趋势图表
- 可日常使用的「身体」管理最小闭环

### v0.2.0 — Work 模块 📋
- 任务 (Task) 数据模型
- 任务 CRUD API
- 基础任务管理 UI

### v0.3.0 — Investment 模块 📋
- 持仓 (Holding) 数据模型
- 投资记录录入
- 收益 / 收益率计算

### v0.4.0 — Knowledge 模块 📋
- 笔记 (Note) 数据模型

- 标签系统

### v0.5.0 — AI 分析层 📋
- 引入 AI 作为分析工具（非数据存储）
- 体重趋势分析

### v1.0.0 — 集成发布 📋
- 四大模块统一导航
- 数据总览 Dashboard
- 文档完善，标记稳定版

## 长期愿景 (Long-term Vision)

- 全模块数据互通：身体状态影响工作效率，投资情绪关联知识学习。
- 完全自托管，数据 100% 自有，可导出到任何格式。
- AI 洞察作为可选项，可插拔。
